조건문

조건문은 조건식의 값이 참인지 거짓인지에 따라 자바스크립트 코드를 제어합니다. 예를 들어 질의 응답 창을 이용해 방문자가 좋아하는 숫자를 입력받았습니다. 이때 입력된 값을 2로 나누어 나머지 값이 0이면 "당신이 좋아하는 숫자는 짝수 입니다."라고 출력하게 하고 , 아닌 경우에는 "당신이 좋아하는 숫자는 홀수입니다."라고 출력하게 한다고 합시다. 이렇게 수를 나눈 나머지 값에 따라 다른 결과를 출력하게 하려면 조건문을 사용해야 합니다. 조건문의 종류에는 if문, else문, 그리고 if else문이 있습니다.

if문

if문은 조건식을 만족(true)할 경우에만 코드를 실행합니다. 조건식은 booline()내장 메서드와 마찬가지로 그 어떤 데이터를 입력해도 true 또는 false를 반환합니다.

* 기본형

if(조건식){
        자바스크립트 코드;
    }

* 예제

var num=10;
if(num<500){
    document.write("hello");
} 
//hello

조건식에 논리형 데이터가 아닌 다른 형이 오는 경우

조건식에 논리형 데이터(true, false)가 아닌 다른 형의 데이터가 입력되어도 true 또는 false로 인식됩니다.

다음값들의 조건식에 입력되면 false를 반환합니다.
0, null, ""(빈문자), undefined

* 예제

var num=0;
if(num){
    document.write(num);
} 
//false

* 예제

<script>
    var min = prompt("당신의 하루 통화량은 몇 분인가요?", "0");
    if(min>=60){document.write("많이 사용하는 편이네요");}
    //  60이상이면 많이 사용하는 편이네요를 출력
</script>

else문

조건식을 만족할(true) 경우와 만족하지 않을(false)의 경우에 따라 실행되는 코드가 달라집니다. 즉, 두 가지 결과가 나올 수 있습니다.

* 예제

<script>
   var num = peompt("당신이 좋아하는 숫자는?", "0");

   if(num % 2 == 0){
       document.write("당신이 좋아하는 숫자는 짝수입니다.");
   } else {
       document.write("당신이 좋아하는 숫자는 홀수입니다.");
   }
   //2의 배수이면 당신이 좋아하는 숫자는 짝수 입니다.
   //나머진 당신이 좋아하는 숫자는 홀수 입니다.를 출력합니다.
</script>

if else문

else if문은 두 가지 이상의 조건식과 정해 놓은 조건을 만족하지 않았을 때 실행되는 코드로 이루어져 있습니다.

두 가지 이상의 조건식과 정해 놓은 조건을 만족하지 않았을 때 실행되는 코드로 이루어져 있습니다.

* 예제

<script>
  var mon = pompt("현재는 몇 월입니까?", "0")

  if(mon>=9 && mon<=11){
      document.write("독서의 계절 가을이네요!");
  } else if(mon>=6 && mon<=8){
    document.write("여행가기 좋은 여름이네요!!");
  } else if(mon>=3 && mon<=5){
      document.write("햇살가득한 봄이네요!");      
  } else {
      document.write("스키의 계절 겨울이네요");
  }
  //9, 10, 11은 독서의 계절 가을이네요!, 6,7,8은 여행가기 좋은 여름이네요!!, 
  3,4,5,햇갈 가득한 봄이네요! 그외는 스키의 계절 겨울이네요를 출력합니다
</script>
