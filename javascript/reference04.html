for()문

for문은 조건식을 만족할 때까지 특정 코드를 반복하여 실행합니다. 용도는 while문과 같지만 더 편해 사용빈도가 더 높습니다.

기본형 :   for(초기값; 조건식; 증감식){
                    조건식의 결과가 참인 동안 반복적으로 실행하고자 하는 실행문;
                }

for()문 문법

<script>

for(var i = 1; i<=10; i++){
    console.log(i);
}

</script>
