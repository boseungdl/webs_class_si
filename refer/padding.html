CSS padding

padding

padding 속성은 요소의 안쪽 여백을 설정합니다.

문법

속성값

  • length(길이) : 패딩을 부동 소수점 숫자 뒤에 절대 단위 지정자( cm , mm , in , pt , pc ) 또는 상대 단위 지정자( em , ex , px )가 오는 값을 지정한다. 기본값은 0px이다.
  • %(퍼센트) : 패딩을 퍼센트로 폭을 적용한다.
  • inherit : 부모 요소로부터 값을 상속 받는다.

정의

  • padding-top속성은 요소의 위쪽 안쪽 여백을 설정
  • padding-right:속성은 요소의 오른쪽 안쪽 여백을 설정
  • padding-bottom속성은 요소의 아래쪽 안쪽 여백을 설정
  • padding-left속성은 요소의 왼쪽 안쪽 여백을 설정
  • padding: 10px 11px 12px 13px; /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */
  • padding: 10px 11px 12px; /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
  • padding: 10px 11px; /* 위,아래:10px 왼쪽,오른쪽:11px */
  • padding: 10px; /* 위,오른쪽,아래,왼쪽 : 10px */
